### Overview

FXCM based program trading model.

### Install

Install the FXCM gym,

```
$ cd algo/gym_fxcm
$ pip install -e .
```

Then install the model requirements,

```
$ cd ../model
$ pip install -r requirements.txt
```

### Build

To build, simply run the model,

```
$ cd algo/model
$ python model.py
```
